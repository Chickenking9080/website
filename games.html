Survival

GAME 01

Protocol Z

A retro-style top-down zombie survival game built with Godot. Scavenge for resources, construct defences, craft weapons, and fend off waves of the undead — alone or with friends.

Survival Multiplayer Singleplayer Godot
View on GitHub 🕹️ itch.io

About the Game

Protocol Z started as a passion project to recreate that classic top-down survival feel with a modern twist. Built entirely in Godot, it features a procedurally assisted map, base-building mechanics, and a crafting system. Jump in solo or invite friends to survive the night together — the difficulty scales with the number of players.

Controls

  • WASD Move
  • Mouse Aim
  • Left Click Shoot / Attack
  • E Interact / Pick up
  • B Build Mode
  • Tab Inventory
  • Space Dodge / Roll
Racing

GAME 02

Extreme Racer

A lightning-fast arcade racer built in just 4 hours during a game jam using Godot. Dodge traffic, push your reflexes to the limit, and chase a high score on a never-ending road.

Racing Arcade Retro Singleplayer Godot
View on GitHub 🕹️ itch.io

About the Game

Extreme Racer was made in 4 hours as a game jam challenge — a testament to what you can build with focus and a clear idea. The game features pixel-art inspired visuals, an escalating difficulty curve, and an addictive score loop. It proved that a fun game doesn't need months of work — just a great core mechanic.

Controls

  • ← → / A D Steer
  • ↑ / W Accelerate
  • ↓ / S Brake
  • Space Handbrake
  • R Restart
